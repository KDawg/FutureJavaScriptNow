ES2015 R&D

Introducing

Use BabelJS to transpile from ES2015 to plain old JS

Install BabelJS in Terminal with this command: npm install -g babel

Running

Compile the entire src directory and output it to the one concatenated file running in Terminal:

babel ./code --out-file main.js --watch

Open the developer tools and look at the Console to see all of the program results. Read the code to see why the run-time output is pertinent.

Discover the J/S BabelJS compiles into main.js.
